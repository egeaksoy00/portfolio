01 — WORKFLOW

Built to remove the repetitive work from internship outreach.

InternGuide turns a raw company list into a structured, reviewable contact dataset through a sequence of controlled processing stages.

01

Upload Companies

Import a structured CSV containing company domains and target locations.

→
02

Discover Contacts

Inspect relevant company pages for public career and contact addresses.

→
03

Filter Emails

Prioritize recruiting, career, HR, application, and general business contacts.

→
04

Export Results

Produce a deduplicated, outreach-ready CSV for the next stage of the workflow.

02 — RUN-BASED SYSTEM

Every operation is tracked as an independent run.

Live polling, structured logs, progress tracking, and persisted state make each run observable from upload to export.

03 — FEATURES

Focused on the parts that usually waste time.

DISCOVERY

Company Discovery

Prepare and process structured company-domain datasets for targeted search regions.

EXTRACTION

Contact Extraction

Search career, contact, jobs, application, and legal pages for relevant public addresses.

FILTERING

Smart Filtering

Keep useful HR and recruiting addresses while suppressing clearly irrelevant categories.

EXPORT

Structured Export

Return deduplicated email datasets in a simplified CSV format for controlled outreach.

04 — UNDER THE HOOD

A small system, built like a system.

FRONTENDReact / Vite

Run controls, progress UI, result inspection.

API LAYERExpress

Run lifecycle, status endpoints, export flow.

ENGINENode.js

Concurrency, page discovery, filtering, deduplication.

PERSISTENCEAtomic JSON

Debounced run-state persistence using temp-write and rename.

Built with concurrency control, domain-level deduplication, network failure handling, and persistent run state.
