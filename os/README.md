# EgeOS

A custom OS-inspired portfolio interface for Ege Aksoy.

This page is designed to run as the monitor/inner-site experience inside the 3D portfolio shell, or standalone at `/os`.

## Files

- `index.html`
- `style.css`
- `script.js`

## Usage

Place these files under your `/os` route or under `static/monitor` if you want to test inside the Henry-inspired 3D monitor iframe.

Then set the iframe source to:

```ts
iframe.src = "https://egeaksoy.net/os";
```
